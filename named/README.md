### Configure DNS Server
- [configuration file](./named.conf) path  `/etc/named.conf`
- [zone file](./dns.zone) path `/var/named/dns.zone`
- [reverse zone file](./dns.zone.rev) path `/var/named/dns.zone.rev`
```
$ chown named:named /var/named -R

```
